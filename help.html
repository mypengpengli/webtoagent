AI File Reader

让网页端 AI 像 IDE 一样读取本地文件

安装本地服务（推荐）

本地服务可以让扩展自动读取你电脑上的文件，无需每次手动授权。

1.确保已安装 Node.js（v14+）
2.在 Chrome 中加载扩展：打开 chrome://extensions，开启开发者模式，点击"加载已解压的扩展程序"，选择扩展目录
3.复制扩展 ID（在扩展卡片上显示的一串字母）
4.编辑 native-host/manifest.json，将 YOUR_EXTENSION_ID_HERE 替换为你的扩展 ID
5.双击运行 native-host/install.bat
6.重启 Chrome，在扩展弹窗中设置工作目录

使用方法

文件树面板

在支持的网站上，页面右下角会出现一个 📁 按钮。点击它打开文件树面板，浏览并点击文件即可将内容插入到聊天输入框。

@ 指令

在聊天输入框中输入 @ 加文件名，会弹出自动补全列表。选择文件后，文件内容会自动插入。

浏览器模式（降级方案）

如果不想安装本地服务，可以使用浏览器的文件系统访问 API：

1.在扩展弹窗中点击"选择文件夹"
2.授权访问你的项目文件夹
注意：浏览器模式每次重启 Chrome 后需要重新授权文件夹访问权限。

常见问题

扩展弹窗显示"未连接"

确认已运行 install.bat，且 native-host/manifest.json 中的扩展 ID 正确。重启 Chrome 后重试。

文件内容没有插入到输入框

部分网站会更新 DOM 结构，导致适配器失效。请在 GitHub 上报告问题。

大文件警告

超过 100KB 的文件会弹出确认提示。超过 1MB 的文件无法读取（防止浏览器卡顿）。
