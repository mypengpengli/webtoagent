WebToAgent

让网页端 AI 读取本地文件并指挥本地智能体工作

安装

1.在 Chrome 中加载扩展：打开 chrome://extensions，开启开发者模式，点击"加载已解压的扩展程序"，选择扩展目录
2.（可选）安装本地服务：双击项目根目录的 安装本地服务.bat，按提示粘贴扩展 ID 即可
3.重启 Chrome，打开任意支持的 AI 网站

使用方法

页面右下角会出现一个 📁 按钮（快捷键 Ctrl+Shift+F）：

文件树点击 📁 打开侧边栏，浏览目录并点击文件插入到聊天框
@ 指令在聊天框输入 @文件名 或 @文件夹名，从下拉列表选择即可插入

两种模式

本地服务模式（推荐）：安装后全自动，打开网页即可用。需要 Node.js。

浏览器模式：无需安装任何东西。打开侧边栏后点击「选择文件夹」授权即可。每次重启浏览器需重新授权。

支持的网站

  • 通义千问 (chat.qwen.ai)
  • ChatGPT (chatgpt.com)
  • Gemini (gemini.google.com)
  • Claude (claude.ai)

快捷键

  • Ctrl+Shift+F — 打开/关闭文件面板
  • @ — 在聊天框中触发文件搜索
  • ↑ ↓ — 在搜索结果中移动
  • Enter 或 Tab — 选中文件
  • Esc — 关闭搜索

卸载

扩展：在 chrome://extensions 中移除

本地服务：双击 卸载本地服务.bat
