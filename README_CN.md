# WebToAgent

English | [中文](./README_CN.md)

一个 Chrome 扩展，让网页端 AI 能读取本地文件，并能指挥本地编程智能体工作——把最强的前沿模型变成你本地工具的项目经理。

## 为什么做这个插件？

**用最强的模型指导便宜的模型干活。**

网页端 AI（ChatGPT、Claude、Gemini）永远跑的是最新、最强的模型——联网、有工具、超大上下文。但 API 端的编程智能体（比如 Claude Code 配 Qwen-27B、DeepSeek 等小模型）运行成本低得多。

有了 **Bridge** 功能，你可以：
- 让网页端大模型（GPT-4o、Claude Opus、Gemini）当**架构师**，出方案、做决策
- 让本地小模型当**执行者**，改代码、跑测试、处理文件
- 大模型给高层指令，小模型听指挥干活
- 效果：比小模型单独做项目好得多，成本只是大模型的零头

**就算不用 Bridge 写代码**，这个插件日常也很好用：
- 点一下文件 → 内容直接插入聊天框，不用复制粘贴
- 点一下图片 → 自动上传为附件，不用手动拖拽
- 点一下文件夹 → 整个目录一次性加载
- 告别复制粘贴、手动上传的繁琐操作

## 功能

**文件访问**
- 文件树侧边栏 — 浏览目录，点击插入
- @ 指令 — `@文件名` 模糊搜索，`@文件夹名` 批量加载
- 图片/二进制上传 — 点击自动模拟拖拽上传
- 文件自动监听 — IDE 里改了文件，索引自动更新
- .gitignore 支持 — 自动过滤无关文件

**Bridge（网页 AI ↔ Claude Code）**
- 网页 AI 回复 → 自动发送给本地 Claude Code
- Claude Code 执行（改文件、跑命令）→ 结果发回网页 AI
- 侧边栏实时进度日志（每个工具调用都能看到）
- 全自动或手动确认两种模式
- 无超时限制，跑多久都行，你手动控制停止

**交互体验**
- 键盘快捷键 `Ctrl+Shift+F`
- 📁 按钮可拖动，带状态指示灯
- 插入历史 + 一键撤销
- 快捷指令模板
- 跨标签页文件索引同步

## 两种模式

| | 本地服务模式 | 浏览器模式 |
|---|---|---|
| 体验 | 全自动，打开网页即可用 | 每次重启浏览器需重新授权 |
| 安装 | Node.js + 双击安装脚本 | 零安装 |
| 原理 | 扩展通过本地 Node.js 程序读取文件 | 浏览器自带的文件系统 API |
| 适合 | 日常使用、Bridge 功能、切换项目 | 临时用一下 |

本地服务不可用时自动降级到浏览器模式。

## 安装

### 第一步：加载扩展

1. 打开 Chrome → `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」→ 选择本项目文件夹

### 第二步（可选）：安装本地服务

需要 [Node.js](https://nodejs.org) v14+

**双击项目根目录下的 `安装本地服务.bat`**

按提示粘贴扩展 ID → 完成。重启 Chrome。

> 不装本地服务也能用 — 打开 AI 网站后点 📁，在侧边栏选择文件夹授权即可。

## 使用方法

### 文件操作

| 操作 | 效果 |
|------|------|
| 点击 📁（或 `Ctrl+Shift+F`） | 打开/关闭文件树侧边栏 |
| 点击文件 | 以 markdown 代码块格式插入 |
| 点击图片/PDF | 自动上传为附件 |
| 输入 `@文件名` | 模糊搜索 → 选中 → 插入 |
| 输入 `@文件夹名` | 批量加载整个目录 |
| 拖动 📁 按钮 | 移到页面任意位置 |

### Bridge（网页 AI ↔ 本地 Claude Code）

1. 打开侧边栏 → 点击 **Bridge: 启动**
2. 正常和网页 AI 对话（给它一个编程任务）
3. AI 回复后，Bridge 自动把回复发给本地 Claude Code
4. Claude Code 执行任务（侧边栏实时显示进度）
5. 结果出现在侧边栏 → 点"发送到网页"发回去（或开启自动发送）
6. 网页 AI 看到结果，给出下一步指令 → 循环继续
7. 随时点 **停止** 结束

**使用技巧：**
- 勾选"全自动发送"实现完全自主运行
- 勾选"显示 CMD 调试窗口"可以看到 Claude Code 的终端
- 侧边栏日志实时显示每一步（💭 思考、🔧 工具调用、✓ 结果）

## 常见问题

**Q: Bridge 功能需要安装 Claude Code 吗？**

需要。全局安装：`npm install -g @anthropic-ai/claude-code`，然后运行一次 `claude login` 登录。

**Q: 可以用便宜的小模型当执行者吗？**

可以。在 Claude Code 的设置中配置使用任意模型（Qwen-27B、DeepSeek 等）。网页 AI 提供智慧，本地模型只管执行。

**Q: 更新了代码需要重新安装吗？**

不需要。在 `chrome://extensions` 点扩展卡片上的刷新图标（↻），然后刷新 AI 网站页面即可。

**Q: Bridge 好像卡住了？**

看侧边栏日志。如果长时间没有活动，可能是 Claude Code 在等待权限确认。打开终端看看是否有 `claude` 进程在运行。
