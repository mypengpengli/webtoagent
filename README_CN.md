# AI File Reader

English | [中文](./README_CN.md)

让网页端 AI（通义千问、ChatGPT、Gemini、Claude）像 IDE 一样自动读取本地文件。

## 功能亮点

- **文件树侧边栏** — 浏览项目目录，点击文件直接插入聊天框
- **@ 指令** — 输入 `@文件名` 模糊搜索，`@文件夹名` 批量加载整个目录
- **文件自动监听** — 在 IDE 里改了文件，索引自动更新，无需手动刷新
- **.gitignore 支持** — 自动读取项目忽略规则，不显示无关文件
- **快捷指令模板** — 保存常用 prompt，一键插入
- **键盘快捷键** — `Ctrl+Shift+F` 开关文件面板
- **按钮可拖动** — 📁 按钮拖到任意位置，自动记忆
- **状态指示灯** — 绿/黄/红小圆点，一眼看出连接状态
- **跨 tab 同步** — 多个标签页共享文件索引，不重复扫描
- **插入历史 + 撤销** — 看到本次会话插入了哪些文件，支持一键撤销

## 两种模式

| | 本地服务模式 | 浏览器模式 |
|---|---|---|
| 体验 | 全自动，打开网页即可用 | 每次重启浏览器需重新授权 |
| 安装 | Node.js + 双击安装脚本 | 零安装 |
| 原理 | 扩展通过本地 Node.js 程序读取文件 | 浏览器自带的文件系统 API |
| 适合 | 长期使用、频繁切换项目 | 临时用一下、不想装东西 |

两种模式功能完全一样。本地服务不可用时自动降级到浏览器模式。

## 安装

### 第一步：加载扩展

1. 打开 Chrome → `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」→ 选择本项目文件夹

### 第二步（可选）：安装本地服务

需要 [Node.js](https://nodejs.org) v14+

**双击项目根目录下的 `安装本地服务.bat`**

按提示粘贴扩展 ID → 完成。重启 Chrome 即可。

> 不装本地服务也能用 — 打开 AI 网站后点 📁，在侧边栏选择文件夹授权即可。

## 使用方法

| 操作 | 效果 |
|------|------|
| 点击 📁（或 `Ctrl+Shift+F`） | 打开/关闭文件树侧边栏 |
| 点击文件树中的文件 | 将文件内容插入聊天输入框 |
| 输入 `@文件名` | 模糊搜索 → 选中 → 插入 |
| 输入 `@文件夹名` | 批量加载整个目录（最多 50 个文件 / 5MB） |
| 拖动 📁 按钮 | 移到页面任意位置，不挡内容 |
| 快捷指令（侧边栏底部） | 一键插入保存的常用 prompt |

文件内容以 markdown 代码块格式插入，自动识别语言。

### 状态指示灯

📁 按钮右下角的小圆点表示连接状态：
- 🟢 绿色 — 本地服务已连接
- 🟡 黄色 — 浏览器模式
- 🔴 红色 — 未配置（点击设置）

### 插入历史

侧边栏顶部显示「已添加 (N)」— 本次会话插入的文件列表。点击「撤销」可移除对应文件。

### 快捷指令

侧边栏底部的「快捷指令」区域：
- 预置了 3 个常用指令（分析代码、解释逻辑、重构）
- 点击即插入到聊天框
- 可自定义添加/删除，持久保存

## 支持的网站

## 卸载

- 扩展：在 `chrome://extensions` 中移除
- 本地服务：双击 `卸载本地服务.bat`
